# channels
**Containers of buffered messages for pull-based event handling.**

* Fully typechecked.
* Designed to be ECS-friendly.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=for-the-badge)](LICENSE)
[![pesde](https://img.shields.io/github/v/tag/realthx1/channels?&style=for-the-badge&label=pesde)](https://pesde.dev/packages/realthx1/channels)

 ---

## Installation
> [!WARNING]
> Currently, only the roblox target is supported in pesde, this will change after pesde v0.8 releases.

Using [**pesde**](https://pesde.dev/):
```bash
pesde add realthx1/channels -t roblox
```
